## Response Example

```json
{
  "entityId": 1,
  "id": "tatooine",
  "name": "Tatooine",
  "rotationPeriod": 23,       // Hours per day
  "orbitalPeriod": 304,       // Days per year
  "diameter": 10465,          // Kilometers
  "climate": "arid",
  "gravity": "1 standard",
  "terrain": "desert",
  "surfaceWater": 1,          // Percentage
  "population": 200000,
  "residents": [...],         // People living here
  "films": [...]              // Films featuring this planet
}
```
